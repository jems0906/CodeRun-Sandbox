## Quick Deploy to Vercel

### 1. One-Click Deploy
Click the button above to deploy directly to Vercel.

### 2. Manual Deploy
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy from client directory
cd client
vercel --prod
```

### 3. Environment Variables
Set this in Vercel:
```
REACT_APP_API_URL=https://web-production-4d4a.up.railway.app/api
```
